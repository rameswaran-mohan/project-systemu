# v0.9.76 — R-A13 Stage-3a: the operator-attestation fallback (behind the flag)

Prepares the external-verification net's arming stage. When the net is enforced, an external effect
that **cannot be independently confirmed** — and for which no independent read-back channel was even
available — now surfaces an **operator-attestation card**: the operator reviews the raw evidence and
vouches that the effect really occurred, crediting it. This is the graceful-degradation path so a
genuinely-unverifiable (but real) effect isn't silently stuck. The net still **stays off** — this
machinery is behind an internal control and the actual arming is a separate, deliberate step; nothing
changes in the default mode.

## What ships

- **An operator-attestation fallback for enforced, unverifiable effects.** In the enforced mode, an
  external effect with no independent confirmation and no available read-back channel raises a card
  showing only the raw, redacted evidence (never the agent's own narrative). The operator's "Attest
  occurred" credits the effect; "Dismiss" leaves it uncredited. This is the system's first
  operator-approval-to-credit path, and it never re-runs the effectful tool.
- **Money-moving effects can never be credited by attestation.** Attestation is a weaker signal than
  an independent read-back, so it is admissible for non-money effects only. A money-move is excluded
  at three independent points (it never raises an attest card; and even if one were forced, the credit
  step re-checks and refuses). A money-move is only ever confirmed by an independent, host-pinned,
  provably-fresh read-back.
- **Attestation is a genuine fallback, not a shortcut.** The card appears only when no independent
  read-back channel was available — so a tool can't sidestep independent verification by routing to
  an easier operator click.

## Compatibility & scope

- **No behavior change with the net off (the default):** the card, its resume, and the credit are all
  gated on the enforced mode; the default and shadow modes are byte-identical to before. The full
  suite (5495 tests) is green. A six-lens adversarial review confirmed the money-move exclusion holds
  and fixed one double-submit edge case (an attested single-objective run with no completed
  predecessor) before release.
