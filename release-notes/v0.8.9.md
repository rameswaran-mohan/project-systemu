# v0.8.9 — Console refinements

Density + ergonomics pass on the v0.8.8 Console.

## What's new
- **Collapsible sections** — Overview (tiles), Pending Actions, Events, More are
  each collapsible and **collapsed by default** on landing. The operator sees a
  compact stack of section headers and expands what they need.
- **Timestamped, newest-first event streams** — both the Supervisor live-events
  pane and the Events Log pane now show an `HH:MM:SS` prefix per row and render
  newest-at-top.
- **Redirect arrows** — the Supervisor Live Events header has a `→` to
  `/systemu-chat` (full supervisor feed); the Events Log header has a `→` to
  `/insights?tab=events`.
- **Smaller heading fonts + tighter spacing** so more fits on the landing screen.

## Backward compatibility
- No route or API changes. `/` still renders the Console.
- The `/insights` Events tab and `/systemu-chat` feed are unchanged (the Console
  panes just deep-link to them).

## Files changed
New: `tests/test_v0_8_9_console.py`, `release-notes/v0.8.9.md`
Modified: `components/live_events_pane.py` (timestamp + newest-first +
`_format_event_time`/`_display_order`), `pages/notifications_page.py`
(`build_events_log_pane` timestamp + newest-first), `pages/console.py`
(collapsible sections + arrows + fonts), version files.

## Migration
`pip install --upgrade systemu`. No config changes.
