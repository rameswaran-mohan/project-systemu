# v0.10.16 — Broken items on your table lead with a fix (R-B2 / T2b)

A card on your **On the table** board that's broken or stale — a server that won't connect, a
credential that needs re-adding — now leads with a one-click way to fix it, instead of leaving you to
hunt for where.

## What ships

- **A repair link on broken/stale cards.** When a table item is `broken` or `stale`, its card shows a
  primary action that deep-links straight to the surface where you fix it: a server or credential →
  Connections/Settings, a tool → the Build page. It's navigation only — the repair happens in the flow
  you land on, and nothing is written from the table (no dual-write).

## Scope

This completes the broken-card action from the table-curation spec (§5.10.c). The remaining T2b
add-side pieces — searching the MCP registry from the palette, deep-linking into forge with context,
live "verify now" health checks, and bulk select — each need new integration flows and are a separate,
focused build. Full suite green.
