## Description

Proficiency in automating the collection of financial market data and compiling it into standardized, dated documentation.

## Procedural Instructions

To automate financial reporting: 1) Use web_screenshot to capture the specific financial chart or data table from the source URL. 2) Use Python's datetime module to generate the required filename based on the current date. 3) Use create_word_doc to assemble the report, embedding the captured image and adding relevant metadata. 4) Ensure the file is saved in the designated directory structure.
