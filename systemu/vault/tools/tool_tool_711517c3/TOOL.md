# file_append

## Description

Append text to an existing file, creating it if it does not exist

## Parameters

- path (string, optional): File path
- content (string, optional): Text to append
- encoding (string, default: utf-8): Encoding

## Returns

- success (boolean)
- path (string)
- error (string)
