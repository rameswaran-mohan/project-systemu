# file_copy

## Description

Copy a file from source to destination

## Parameters

- src (string, optional): Source file path
- dst (string, optional): Destination file path
- overwrite (boolean, default: False): Overwrite destination if it exists

## Returns

- success (boolean)
- dst (string)
- error (string)
