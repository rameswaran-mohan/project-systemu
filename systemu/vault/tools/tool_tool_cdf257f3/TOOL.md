# extract_archive

## Description

Extract a ZIP or TAR archive to an output directory

## Parameters

- archive_path (string, optional): Path to archive file (.zip, .tar, .tar.gz, .tgz)
- output_dir (string, optional): Directory to extract into

## Returns

- success (boolean)
- output_dir (string)
- file_count (integer)
- error (string)
