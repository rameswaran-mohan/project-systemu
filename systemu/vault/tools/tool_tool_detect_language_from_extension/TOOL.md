# detect_language_from_extension

## Description

Detect programming language from a file extension using a built-in mapping dictionary.

## Parameters

- filename (string): Filename with extension (e.g., 'main.py')

## Returns

- success (boolean)
- language (string) — Detected programming language name (e.g., 'Python', 'Unknown')
- error (string)
