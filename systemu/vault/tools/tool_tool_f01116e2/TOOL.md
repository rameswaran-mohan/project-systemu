# format_date

## Description

Parse a date string and reformat it from one format to another

## Parameters

- date_str (string, optional): Date string to convert
- input_format (string, default: %Y-%m-%d): strptime format of input
- output_format (string, default: %m/%d/%Y): strftime format for output

## Returns

- success (boolean)
- result (string)
- error (string)
